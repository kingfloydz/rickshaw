# mjlab 黄包车训练大纲

## 1\. 训练目标

* 在平地上训练黄包车稳定跟踪目标前进速度和转弯。
* 左右牵引点分别施加三维力，策略动作共 6 维。
* 两个牵引点的目标离地高度均为 0.75 m。
* 在满足速度、高度和姿态要求后，尽量减小牵引力幅值及变化率。
* 暂不进行黄包车参数、地面摩擦和传感器随机化。

## 2\. 模型要求

* 地面采用水平 plane，黄包车以正常静止姿态初始化，牵引点高度为0.75m。

## 3\. 动作设计

* 策略输出左右牵引点在车体坐标系下的三维力。
* 将车体坐标系力转换到世界坐标系。
* 保存当前和上两时刻实际施加的力，用于观测和力变化率奖励。

## 4\. 观测设计

Actor 使用 25 维观测：

* 前进速度和转弯指令（车体坐标系）：2 维。
* 车体线速度（车体坐标系）：3 维。
* 车体角速度（车体坐标系）：3 维。
* 投影重力（车体坐标系）：3 维。
* 两牵引点高度：2 维。
* 上一时刻施加的左右三维力：6 维。
* 上上一时刻施加的左右三维力：6 维。

Critic 观测增加两轮角速度、坡度和两轮接触状态。

## 5\. 牵引点高度

* 牵引点高度定义为牵引点到其下方地面在地面法向（直接由坡度计算）上的距离。

## 6\. 奖励设计

\## 总奖励



\\\[

\\begin{aligned}

r\_t={}\&

3.0r\_{v\_x}

+1.5r\_{\\omega\_z}

+0.8r\_h\\\\

\&-0.20c\_{v\_{yz}}

\-0.10c\_{\\omega\_{xy}}

\-0.15c\_{\\text{slip}}

\-0.50c\_{\\text{lift}}\\\\

\&-\\rho\\left(

0.05c\_{\\text{peak}}

+0.08c\_{\\Delta F}

+0.01c\_{\\Delta^2F}

\\right)

\\end{aligned}

\\]



\## 奖励函数



| 名称 | 奖励函数 | 权重 |

|---|---|---:|---|

| `track\_forward\_velocity` | \\(\\exp\[-(v\_x-v\_x^\*)^2/\\sigma\_v^2]\\) | `3.0` | 

| `track\_yaw\_velocity` | \\(\\exp\[-(\\omega\_z-\\omega\_z^\*)^2/\\sigma\_\\omega^2]\\) | `1.5` | 

| `traction\_point\_height` | \\(\\exp\[-\\sum\_i(h\_i-h\_i^\*)^2/\\sigma\_h^2]\\) | `0.8` | 

| `undesired\_linear\_velocity` | \\((v\_y/0.5)^2+(v\_z/0.3)^2\\) | `-0.20` | 

| `roll\_pitch\_rate` | \\((\\omega\_x/1.0)^2+(\\omega\_y/1.0)^2\\) | `-0.10` | 

| `wheel\_slip` | 左右轮滚动误差与侧向滑移平方和 | `-0.15` | 

| `wheel\_lift` | \\((1-c\_L)+(1-c\_R)\\) | `-0.50` | 

| `peak\_force` | 牵引力超过软限制的平方惩罚 | `-0.05` | 

| `force\_continuity` | 相邻时刻实际力变化平方和 | `-0.08` | 

| `force\_second\_difference` | 实际力二阶差分平方和 | `-0.01` | 

| `termination` | 翻倒、轮胎持续离地 | -200 | 



参数：



```text

sigma\_v   = 0.4 m/s

sigma\_yaw = 0.3 rad/s

sigma\_h   = 0.05 m

```



\## 牵引点高度奖励



\\\[

r\_h=

\\exp\\left\[

\-\\frac{(h\_L-h\_L^\*)^2+(h\_R-h\_R^\*)^2}{\\sigma\_h^2}

\\right]

\\]



\## 轮胎离地惩罚



\\\[

c\_{\\text{lift}}=(1-c\_L)+(1-c\_R)

\\]



其中 \\(c\_i=1\\) 表示接触，\\(c\_i=0\\) 表示离地。使用最近 3 个物理子步的接触状态。



\## 力相关奖励



前进与转弯奖励达到80%后，令课程系数 \\(\\rho\\) 从 0 平滑增加到 1。



最大力：



\\\[

F\_{\\text{peak}}

=

\\max\\left(\\|F\_L\\|\_2,\\|F\_R\\|\_2\\right)

\\]



\\\[

c\_{\\text{peak}}

=

\\left\[

\\frac{\\max(0,F\_{\\text{peak}}-10)}

{40}

\\right]^2

\\]

连续性：



\\\[

c\_{\\Delta F}

=

\\frac{

\\|F\_{L,t}-F\_{L,t-1}\\|^2+

\\|F\_{R,t}-F\_{R,t-1}\\|^2

}{

2F\_{\\text{hard}}^2

}

\\]



二阶连续性：



\\\[

c\_{\\Delta^2F}

=

\\frac{

\\sum\_{i\\in\\{L,R\\}}

\\|F\_{i,t}-2F\_{i,t-1}+F\_{i,t-2}\\|^2

}{

2F\_{\\text{hard}}^2

}

\\]



## 7\. 终止条件

* 单回合达到设定时长。
* 车身俯仰或侧倾超过安全角度。
* 轮胎持续离地
